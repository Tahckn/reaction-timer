# Reaction Timer

Reaction tester app

Created with Vue.js and Tauri for Windows app.

 [Website version](https://reaction-timerr.netlify.app/).

```bash
Export the .msi file from zip and install thats it.
```

## Usage
 ```bash
npm run install

npm run tauri build - (for build windows app)
```
